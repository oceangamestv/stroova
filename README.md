# STroova

Веб‑приложение для изучения английского языка с системой учётных записей и упражнениями.

## Как запустить

### Разработка

1. Установите зависимости: `npm install`
2. Запустите фронтенд: `npm run dev` (порт 5173)
3. Запустите API сервер: `npm run server` (порт 3000)
4. Откройте http://localhost:5173 в браузере

### Сборка

1. Соберите проект: `npm run build`
2. Файлы будут в папке `dist/`

## Технологии

- **Frontend**: React 18 + TypeScript + Vite
- **Backend**: Node.js + Express + PostgreSQL
- **Mobile**: Capacitor (Android/iOS)
- **Routing**: React Router DOM

## Что реализовано

### Система учётных записей
- Регистрация и авторизация пользователей
- Профиль пользователя с статистикой
- Система рейтинга пользователей
- Сохранение прогресса на сервере

### Персональный словарь
- Просмотр словаря с фильтрацией и поиском
- Система прогресса изучения слов
- Поддержка британского и американского акцентов
- IPA транскрипция для произношения

### Упражнения
1. **Поиск пары** (`/pairs`) - сопоставление английских слов и переводов
2. **Puzzle Words** (`/puzzle`) - сборка слова из букв
3. **Данетка** (`/danetka`) - угадывание слова по описанию
4. **1 из 3** (`/one-of-three`) - выбор правильного перевода

### Дополнительные возможности
- Темная/светлая тема
- Адаптивный дизайн для мобильных устройств
- Звуковое произношение слов
- Система XP и уровней
- Telegram бот (опционально)
