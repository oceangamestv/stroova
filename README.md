# LinguaMatch — мини‑клон

Веб‑приложение для изучения английского языка с системой учётных записей и двумя типами упражнений.

## Как запустить

1. Откройте `login.html` в браузере (Chrome, Edge, Firefox и т.п.).
2. Зарегистрируйте новый аккаунт или войдите в существующий.
3. Интернет и сервер не требуются — всё работает локально на HTML/CSS/JS.
4. Данные пользователей сохраняются в localStorage браузера.

## Что реализовано

### Система учётных записей
- **Регистрация**: создание аккаунта с логином и паролем (минимум 3 символа для логина, 4 для пароля).
- **Вход**: авторизация по логину и паролю.
- **Выход**: кнопка выхода из аккаунта.
- **Сохранение прогресса**: статистика пользователя (очки, выполненные упражнения) сохраняется автоматически.

### Персональный словарь с прогрессом
- **Отдельная страница словаря**: просмотр всех слов уровня A0 с индивидуальным прогрессом.
- **Система прогресса**: каждое слово может быть изучено от 0% до 100%.
- **Автоматическое обновление**: при правильном ответе в упражнении +1% к знанию слова, при ошибке -1%.
- **Визуализация прогресса**: прогресс-бары и процент изучения для каждого слова.
- **Фильтры и поиск**: фильтрация по акценту (UK/US/both), поиск по английскому или русскому тексту.
- **Статистика**: общее количество слов, изученных слов (100%), средний прогресс.

### Словарь уровня A0
- **330+ базовых слов** английского языка уровня A0
- **Разделение по акцентам**: каждое слово помечено как британский (UK), американский (US) или общий (both)
- **Категории слов**: приветствия, числа, цвета, семья, еда, дни недели, месяцы, животные, части тела, одежда, дом, школа, природа, транспорт, глаголы и многое другое
- **Визуальные метки**: слова с британским или американским акцентом отмечены флагами на карточках

### Упражнения
1. **Поиск пары**: сопоставление английских слов и русских переводов.
   - Интерактивные карточки с визуальной обратной связью.
   - Показ акцента (UK/US) на английских карточках.
   - **Звуковое произношение**: при правильном ответе слово произносится на английском с учётом акцента (британский/американский).
   - **Звук ошибки**: при неправильном ответе воспроизводится короткий нейтральный звук.
   - Случайный выбор слов из словаря A0 при каждом запуске.
   - Подсчёт очков: +10 за правильную пару, −3 за ошибку.
   - Прогресс‑бар и счётчик найденных пар.

2. **Сборка слова из пазлов**: составление английского слова из букв.
   - Показывается русский перевод.
   - Буквы перемешаны, нужно собрать слово.
   - Случайный выбор слова из словаря A0 при каждом запуске.
   - Поддержка ввода с клавиатуры: нажимайте буквы на клавиатуре для быстрого заполнения.
   - Подсчёт очков: +15 за правильный ответ, −5 за ошибку.

### Интерфейс
- Современный дизайн в стиле Duolingo.
- Адаптивная вёрстка для мобильных устройств.
- Модальные окна с результатами упражнений.
- Переключение между типами упражнений.
