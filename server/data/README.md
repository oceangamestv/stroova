# Данные для сида словаря

Положите сюда файл **dictionary_A0_A2_2000_unique_freq_register.csv** — тогда по умолчанию `npm run seed` будет загружать из него ~2000 слов (редкость и регистр из CSV).

Колонки CSV: `id`, `language_id`, `en`, `ru`, `accent`, `level`, `frequency_rank`, `rarity`, `register`, `ipa_uk`, `ipa_us`, `example`, `example_ru`.
