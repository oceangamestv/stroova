# Деплой STroova

**Боевой сервер:** https://stroova.ru (подключение по SSH: `ssh root@stroova.ru`). Локальная разработка без изменений: `npm run dev` (5173) + `node server/index.js` (3000).

## Кратко

- **Первый раз:** выполнить шаги из DEPLOY-УБУНТУ-ПОШАГОВО.md (подключение: `ssh root@stroova.ru`).
- **Обычное обновление:** на компьютере `git push` → на сервере `cd ~/stroova && ./deploy.sh`.
- **Автодеплой:** в [DEPLOY-ОБНОВЛЕНИЕ.md](DEPLOY-ОБНОВЛЕНИЕ.md) раздел «Автодеплой по push». Секрет `SSH_HOST` = `stroova.ru`.

Данные хранятся в PostgreSQL на сервере; при обновлении БД не пересоздаётся.
